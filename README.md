## Sobre mim

Desenvolvedor Full Stack com **+5 anos em produção**, construindo sistemas do backend ao mobile.

Atualmente na **Orange Black**, onde lidero o desenvolvimento de um SaaS multi-tenant para gestão de motéis — do backend NestJS com lógica de precificação complexa ao app React Native com design system próprio.

Foco em arquitetura que escala, código que o time consegue evoluir, e interfaces que fazem sentido para quem usa.
